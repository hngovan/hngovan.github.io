# Keeb Log

Keeb Log is an Astro blog for Vietnamese notes about custom mechanical keyboards: build logs, switch notes, keycap profiles, layout decisions, and desk setup experiments.

This site is cloned from [Snow0406/hyuki.dev](https://github.com/Snow0406/hyuki.dev), built with [Astro](https://astro.build/), [Tailwind](https://tailwindcss.com/), and [shadcn/ui](https://ui.shadcn.com/). The original project is based on [astro-erudite](https://github.com/jktrn/astro-erudite).

## Development

```bash
bun install
bun run dev
```

Build check:

```bash
bun run build
```

## Content

- Blog posts live in `src/content/blog`.
- Keyboard build/project notes live in `src/content/projects`.
- Authors live in `src/content/authors`.
